# EJSBASIC (LAB06)

This folder contains the LAB 6 project for EJS basics.

## Included
- `src/` (TypeScript source)
- `views/` (EJS templates)
- `public/` (static assets)
- `package.json`, `tsconfig.json`
- `answers.md` (answers to lab questions)

## Run locally
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open `http://localhost:3000`

## Notes
- Do **not** commit `node_modules/` (a `.gitignore` is provided).
- The EJS views folder is set in `src/app.ts` as `path.join(__dirname, '..', 'views')`.
